# resume
A repository containing my resume and the LaTex files to build it. Forked from https://github.com/posquit0/Awesome-CV

To build: `xelatex resume.tex`

## Dependencies
- enumitem
- ragged2e
- xcolor
- xifthen
- ifmtarg
- etoolbox
- setspace
- fontspec
- unicode-math
- fontawesome5
- sourcesanspro
- xkeyval
- tcolorbox
- pgf
- environ
- parskip
