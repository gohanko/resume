# resume
A repository containing my resume and the LaTex files to build it. Forked from https://github.com/posquit0/Awesome-CV

To build: `xelatex resume.tex`

## Getting Started (Fedora)

1. Install TextLive: `sudo dnf install texlive-scheme-basic`
2. Install Dependencies: `sudo dnf install 'tex(enumitem.sty)' 'tex(ragged2e.sty)' 'tex(xcolor.sty)' 'tex(xifthen.sty)' 'tex(ifmtarg.sty)' 'tex(etoolbox.sty)' 'tex(setspace.sty)' 'tex(fontspec.sty)' 'tex(unicode-math.sty)' 'tex(fontawesome5.sty)' 'tex(sourcesanspro.sty)' 'tex(xkeyval.sty)' 'tex(tcolorbox.sty)' 'tex(pgf.sty)' 'tex(environ.sty)' 'tex(parskip.sty)' 'tex(roboto.sty)' 'tex(tikzfill.image.sty)'`
